# Space Oppressor

This is a 2D Space Invader-like game made with Pygame, as a side project for my own learning of Python.

## Features

- Movable player
- Enemies
- Player that can shoot projectiles
- Player can die
- Enemies can die
- Score and health system
- Score and health UI

## Technologies

- Python
- Pygame

## How to Run

1. Clone the repository: 
```bash
git clone https://github.com/Itamar1567/Space_Oppressor.git
cd Space_Oppressor
```
2. Create a virtual environment (Optional):
```bash
python -m venv venv
source venv/bin/activate  # On Linux/Mac
venv\Scripts\activate     # On Windows
```
3. Install dependencies:
```bash
pip install pygame
```
4. Run the game
```bash
python main.py
```
